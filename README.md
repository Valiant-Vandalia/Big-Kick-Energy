# Big Kick Energy

Official team website for Big Kick Energy, a Charleston Kickball Association team in Charleston, West Virginia.

## Website

The site is deployed automatically to GitHub Pages whenever the `main` branch is updated.

## Content

- 2026 record and Wednesday standings snapshot
- Team identity and values
- Game-day information
- Team photo gallery
- Official Charleston Kickball Association links

Built as a lightweight, accessible static website with no tracking and no required JavaScript.
